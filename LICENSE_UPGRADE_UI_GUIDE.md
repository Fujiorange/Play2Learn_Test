## Color Scheme

### Buttons
- **Primary (Upgrade, Select Plan, Complete Payment)**: Green gradient (#10b981 to #059669)
- **Secondary (Cancel, Back)**: Gray (#f3f4f6)
- **Back to Dashboard**: Light gray with border

### License Badges
- **Free Trial**: Green gradient
- **Paid Plans**: Blue gradient

### Usage Indicators
- **< 70%**: Green (#10b981)
- **70-90%**: Orange (#f59e0b)
- **> 90%**: Red (#ef4444)

### Form States
- **Default**: Gray border (#e5e7eb)
- **Focus**: Green border with shadow (#10b981)
- **Error**: Red border (#ef4444)

### Messages
- **Success**: Green background
- **Error**: Red background  
- **Info**: Blue background
- **Warning**: Yellow background

## Responsive Design

### Desktop (> 768px)
- License plans displayed in a grid (up to 3 columns)
- Payment form side-by-side for expiry and CVV
- Full width for all elements

### Mobile (< 768px)
- License plans stacked vertically
- Payment form stacked vertically
- Full width buttons
- Larger touch targets

## Accessibility

- All form fields have labels
- Error messages are clearly visible
- Color contrast meets WCAG guidelines
- Keyboard navigation supported
- Screen reader friendly

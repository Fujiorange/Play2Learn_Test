## Color Scheme

- **Primary (Success)**: Green (#10b981)
- **Error**: Red (#ef4444)
- **Border**: Light gray (#e5e7eb)
- **Background**: White (#ffffff)
- **Text Primary**: Dark gray (#1f2937)
- **Text Secondary**: Medium gray (#6b7280)
- **Inactive**: Light gray (#f3f4f6)

## Responsive Breakpoints

- **Desktop**: 768px and above (3-column grid)
- **Tablet**: 768px (2-column grid)
- **Mobile**: Below 768px (1-column grid)

## Hover Effects

- Cards lift up slightly on hover
- Buttons change color and shadow on hover
- Template buttons show green border on hover

## Accessibility Features

- All form fields have labels
- Required fields marked with *
- Help text for complex fields
- Keyboard navigation support
- ARIA labels on buttons
- Color contrast meets WCAG standards
